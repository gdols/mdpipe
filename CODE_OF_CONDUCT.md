# Code of Conduct

This is a small project, so the rules are short: be kind, be patient, and assume good intentions.

Disagreeing about code is fine (and useful!). Being rude, dismissive or personal about it is not.
Harassment or discrimination of any kind won't be tolerated.

If someone's behaviour is making the project unpleasant, email me at **guille@gdols.dev** and I'll deal
with it, anything from a friendly word to a ban, depending on what's warranted.
